Mail
====

**Mail** is a component which allow to create/send email easily whatever mailer
mecanism you use (SwiftMailer...).

Installation
------------

Using Composer, just require the `ddd/components` package:

``` javascript
{
    "require": {
        "ddd/components": "dev-master"
    }
}
```
